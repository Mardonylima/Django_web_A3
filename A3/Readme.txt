# Sistema de Cadastramento de Clientes

Este projeto é um sistema básico de cadastramento de clientes desenvolvido em Django. 
Ele permite realizar as operações de adicionar, editar, listar e deletar clientes, 
com foco em qualidade de software, controle de versão e estratégias de teste.

---

## **Funcionalidades**
- **Adicionar Cliente:** Registre novos clientes com informações básicas como nome, e-mail, telefone e endereço.
- **Editar Cliente:** Atualize os dados de clientes existentes.
- **Listar Clientes:** Visualize uma lista completa de clientes com opções de busca e ordenação.
- **Deletar Cliente:** Remova clientes do sistema.

---

## **Tecnologias Utilizadas**
- **Backend:** Django
- **Frontend:** HTML, CSS, Bootstrap
- **Banco de Dados:** SQLite (padrão) com possibilidade de migração para PostgreSQL/MySQL
- **Controle de Versão:** Git e GitHub
- **Testes:** Pytest ou Unittest
